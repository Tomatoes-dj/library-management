# library-management
此案例为熟悉Vue常用特性建立的。
主要逻辑如下： 
1. 图书列表  
· 实现静态列表效果  
· 基于数据实现模板效果  
· 处里每行的操作按钮   
2. 添加图书  
· 实现表单的静态效果  
· 添加图书表单域数据绑定  
· 添加按钮事件绑定  
· 实现添加业务逻辑  
3. 修改图书   
· 修改信息填充到表单  
· 修改后重新提交表单 
· 重用添加和修改的方法  
4. 删除图书  
· 删除按钮绑定事件处理方法  
· 实现删除业务逻辑  
5. 常用特性应用场景  
· 过滤器（格式化日期）  
· 自定义指令（获取表单焦点）  
· 计算属性（统计图书数量）  
· 侦听器（验证图书存在性）  
· 生命周期（图书数据处理）
